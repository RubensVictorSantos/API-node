Get Todos cliente registrados no banco

http://localhost:3333/cliente

Para retorna um JSON com todos os clientes registrados no banco, precisamos acrescentar a palavra cliente.

Example Request
    
    var settings = {
        "url": "http://rubens-pc:3333/cliente",
        "method": "GET",
        "timeout": 0,
        };
        
        $.ajax(settings).done(function (response) {
        console.log(response);
    });

Get Cliente pelo ID

http://localhost:3333/cliente/1

Retorna um JSON com todas as informações do cliente especificado pelo id, precisamos acrescentar o número do id na url depois de cliente/.

Example Request
    
    var settings = {
        "url": "http://127.0.0.1:3333/cliente/1",
        "method": "GET",
        "timeout": 0,
        };
        
        $.ajax(settings).done(function (response) {
        console.log(response);
    });
